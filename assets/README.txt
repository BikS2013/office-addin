Office Add-in Icon Assets
========================

Place the following icon files in this directory:

  - icon-16.png  (16x16 pixels)   - Small ribbon icon
  - icon-32.png  (32x32 pixels)   - Standard ribbon icon
  - icon-80.png  (80x80 pixels)   - High resolution icon / store listing

These icons are referenced by the manifest.xml and displayed in the Word ribbon
and task pane header. They should be PNG format with transparent backgrounds.

For development, placeholder icons can be generated using any image editor
or online tool. For production, use icons that follow Microsoft's Office Add-in
icon design guidelines:
https://learn.microsoft.com/en-us/office/dev/add-ins/design/add-in-icons
